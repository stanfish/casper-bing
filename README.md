## Description

Login to a microsoft account and check current reward point daily


## Requirement

CasperJS

[http://casperjs.org/]


## Install

`npm install`


## Run

`casperjs index.js`


## Log 

`tail myCasper.log -F`


## Daily Cron Job

`crontab -e`

Add

`0 6 * * * /PATH/script/dailyscript.sh`
